# Bullet Journal — Wireframe v2 (Alarmy-Inspired Redesign)

> 알라미 앱의 핵심 UX 패턴을 적용한 불렛저널 리디자인
> - 점진적 정보 공개 (Progressive Disclosure)
> - 게이미피케이션 (스트릭, 달성률, 성공 피드백)
> - 리포트 피드백 루프 (기록 → 분석 → 동기부여)
> - 4-Level 정보 계층

---

## 1. 정보 설계 (IA) — 6탭 → 5탭 재구성

### AS-IS (6탭)
```
오늘 | 주간 | 습관 | 월간 | 반기 | 가치
```
- 문제: 탭 과부하, 피드백 루프 없음, 계층 연결 약함

### TO-BE (5탭)
```
오늘 | 플랜 | 트래커 | 리포트 | 마이
```

| 탭 | 역할 | 통합 대상 |
|-----|------|----------|
| **오늘** | 오늘 하루 대시보드 + 타임블록 | 기존 Daily |
| **플랜** | 주간·월간 계획 통합 뷰 | 기존 Weekly + Monthly |
| **트래커** | 습관 트래킹 + 스트릭 | 기존 Habit (게이미피케이션 강화) |
| **리포트** | 통계·분석·트렌드 (**신규**) | 알라미의 리포트 탭 차용 |
| **마이** | 가치·반기 목표·설정 통합 | 기존 Value + Yearly |

## 3. 게이미피케이션 시스템 (Alarmy-Inspired)

알라미가 "기상 성공"을 게임처럼 만든 것처럼, 불렛저널도 "하루 완성"을 게임화한다.

### 3-1. 스트릭 (연속 달성)

```
┌─────────────────────────────┐
│  🔥 12일 연속 달성!           │
│                              │
│  월 화 수 목 금 토 일         │
│  ●  ●  ●  ●  ◐  ○  ○        │   이번 주 진행도
│                              │
│  "12일 연속이에요! 대단해요"  │   격려 메시지
└─────────────────────────────┘
```

**달성 조건** (하루 완성 = 아래 중 3개 이상):
- 타임블록 4시간 이상 체크
- 습관 50% 이상 완료
- 저녁 리뷰 작성

```
│                              │

```
